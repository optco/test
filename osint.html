OSINT & Cybersecurity Toolkit

Curated collection of open-source intelligence tools, reconnaissance frameworks, and cybersecurity utilities for ethical hacking, threat intelligence, and digital investigations. All tools are for educational and authorized security testing purposes only.

🔍

Search Engines & Reconnaissance

Shodan
Active
→ https://www.shodan.io

Search engine for Internet-connected devices. Find servers, cameras, printers, IoT devices, and vulnerable systems exposed to the internet. Essential for infrastructure reconnaissance.

Censys
Active
→ https://censys.io

Attack surface management platform. Discover and analyze internet-connected assets, certificates, and hosts. Advanced search capabilities for security researchers.

Intelligence X
Active
→ https://intelx.io

Search engine and data archive. Search through leaks, data breaches, darknet, and public records. Powerful OSINT tool for finding historical data and compromised credentials.

FOSSA
Active
→ https://fossa.com

Open source dependency tracking and security. Identify vulnerabilities in software supply chains and third-party components.

Network Scanning & Enumeration

Maltego
Active
→ https://www.maltego.com

Graph-based link analysis and data mining tool. Visualize relationships between people, companies, domains, IPs, and infrastructure. Community edition available.

Alternative: SpiderFoot (open-source alternative)
SpiderFoot
Active
→ https://www.spiderfoot.net

Open-source intelligence automation tool. Collect data from over 100 sources for IP addresses, domains, emails, and more. CLI and web interface available.

Recon-ng
Active
→ GitHub Repository

Full-featured web reconnaissance framework written in Python. Modular architecture with numerous modules for gathering information from public sources.

Alternative: BBOT (modern alternative)
BBOT
Active
→ GitHub Repository

Big Bad Offensive Security Tool. Fast, recursive OSINT scanner and framework. Modern alternative to Recon-ng with improved speed and automation.

CloudFox
Active
→ GitHub Repository

AWS and cloud infrastructure enumeration tool. Automate cloud security assessments and identify misconfigurations in AWS, Azure, and GCP environments.

👤

Social Media & Identity OSINT

Sherlock
Active
→ GitHub Repository

Find usernames across social networks. Search for accounts by username across 300+ social media platforms. Essential for digital footprint analysis.

PhoneInfoga
Active
→ GitHub Repository

Advanced phone number reconnaissance tool. Gather information about phone numbers using various search engines and APIs. Web interface and CLI available.

⚡

Vulnerability Scanning & Testing

Nuclei
Active
→ GitHub Repository

Fast, customizable vulnerability scanner. Template-based scanning for web applications, networks, and cloud infrastructure. Large community template library.

Burp Suite
Active
→ https://portswigger.net/burp

Industry-standard web application security testing tool. Intercept, analyze, and modify web traffic. Community (free) and Professional editions available.

Free Alternative: Caido (lightweight alternative)
Caido
Active
→ https://caido.io

Lightweight web security testing platform. Modern alternative to Burp Suite with focus on speed and simplicity. Web proxy and vulnerability scanner.

BloodHound
Active
→ GitHub Repository

Active Directory attack path analysis. Visualize and identify complex attack paths in AD environments. Essential for red team operations and AD security assessments.

Evilginx3
Active
→ GitHub Repository

Advanced phishing framework with MFA bypass capabilities. Create man-in-the-middle phishing attacks that capture credentials and session cookies.

Analysis & Decoding Tools

CyberChef
Active
→ https://gchq.github.io/CyberChef

"Cyber Swiss Army Knife" for encoding, decoding, and data analysis. Hundreds of operations for cryptography, compression, and data transformation. Runs in browser.

EXIF Search
Active
→ https://exif.tools

Extract metadata from images and documents. Find GPS coordinates, camera info, software used, and hidden data in file metadata.

Alternative: ExifData.com
🌑

Dark Web Search Engines

Ahmia
Active
→ https://ahmia.fi

Search engine for Tor hidden services. Index .onion sites with focus on safety and removing abuse content. Clearnet interface to search dark web content.

Torch
Unstable
→ Tor Hidden Service (.onion)

One of the oldest Tor search engines. Index thousands of .onion sites. Note: Requires Tor Browser to access. Availability varies frequently.

Alternatives: Ahmia (more stable), Dark.fail (directory)
Grams
Offline
→ Service No Longer Available

Dark web search engine (discontinued). Was popular for searching darknet markets and hidden services.

Alternatives: Ahmia, Intelligence X
